# PCG Nodes Reference

PCG Graphで使用可能なUPCGSettings派生ノードの一覧です。分類は各クラスの `GetType()` 実装に基づき、ローカライズ済みのノード名を優先的に記載しています。（日本語名が未翻訳の場合は英語名を表示）

- 総ノード数: 197
- データ生成元: ソースコード内のUPCGSettings派生クラス（テスト用クラスを除外）
- 日本語訳は `Engine/Content/Localization/Engine/ja/Engine.po` を参照し、存在しない場合は英語名を流用しています
- 説明はノードのツールチップ（`GetNodeTooltipText()`）および必要に応じて手動追記した内容を基にしています

> 各ノードの詳細な設定項目は `Docs/PCG/Nodes/` 配下の個別ファイルに移動しました。ノード名をクリックすると該当ドキュメントへ遷移します。
